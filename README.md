# telegram chat viewer

одностраничный вьювер экспортированной переписки Telegram (HTML-экспорт) с поиском, медиа-панелью, темами и акцентными цветами. работает как самостоятельный HTML-файл в браузере или как десктоп-приложение на Tauri.

## возможности
- открытие экспорта Telegram (result.json / HTML-экспорт) без сервера
- поиск по сообщениям с подсветкой совпадений
- отдельная панель медиа с сеткой вложений
- красивая персонализация
- локализация RU/EN

## установка

### вариант 1 — просто открыть в браузере
скачай `index.html` и открой его в браузере — ничего устанавливать не нужно.

### вариант 2 — десктоп-приложение (Windows)
скачай готовый установщик из раздела **Releases** этого репозитория.

## сборка из исходников
```bash
cd tauri-project
npm install
npm run tauri build
```
готовый `.exe`/`.msi` появится в `tauri-project/src-tauri/target/release/bundle/`.

для запуска без сборки инсталлятора:
```bash
cd tauri-project
npm install
npm run dev
```

## структура репозитория
```
telegram_chat_viewer.html   - самостоятельная версия вьювера
icons/                      - иконки приложения
tauri-project/              - десктоп-обёртка на Tauri
  src/index.html            - тот же вьювер внутри приложения
  src-tauri/                 - код Rust-обёртки и конфиг Tauri
```

## стек
чистые HTML/CSS/JS без фреймворков + Tauri (Rust) для десктоп-обёртки.
